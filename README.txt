This program calculates coefficients in chemical reaction equations.
The program is implemented in the Java language.

Current Maven build builds GUI version of application.
You can create console version using class ConsoleVersion.

=======

Представленная программа производит вычисление коэффициентов в химических уравнениях реакции.
Программа реализована на языке Java.

При текущей настройке Maven строит версию программы с графическим интерфейсом.
Консольная версия программы доступна из класса ConsoleVersion.
